| 标题   | 内容                                       |
| ---- | ---------------------------------------- |
| 类型   | 通用                                       |
| 支持布局 | responsive,fixed-height,fill,container,fixed |
| 所需脚本 | https://mipcache.bdstatic.com/static/v1/mip-jt-quote-rt/mip-jt-quote-rt.js |
